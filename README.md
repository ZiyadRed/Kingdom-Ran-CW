# Kingdom Ran EN – Souha Skill Simulator

An English fan translation of the Japanese skill simulator at [pirock55.work](https://pirock55.work/souha-skill-archive/), for the mobile game **Kingdom Ran** (キングダム乱).

---

## Features
- **Skill Archive** – Browse all translated generals, search by name (EN/JP), filter by country and skill type
- **Party Builder** – Pick up to 6 generals for a Souha battle party
- **Activation Order** – Preview skill sequence (full simulation logic coming soon)

---

## Quick Start

Make sure you have **Node.js 18+** installed.
